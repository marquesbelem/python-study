# Bonus: Projeto de calcular fator R
Fator R = Receita Bruta (12 meses) / Folha de Pagamento (12 meses)
​
### Projeto: Calculadora de Fator R 
**Objetivo:**
- Saber qual Fator R a empresa se encaixa partindo das informações de receita Bruta e Folha de Pagamento
- Receita Bruta: Será usado excel estraido do proprio site "Nota de 1milhão". 
- Folha de Pagamentos: PDFs gerado pela contabilidade. 

**Bibliotecas**
- `os:` Interage com o sistema operacional (pastas, caminhos).
- `re:` Trabalha com expressões regulares (padrões de texto avançados)..
- `datetime:` Registra data/hora no log.
